🚦 Accident Prediction System
📌 Project Overview

The Accident Prediction System is a Machine Learning project designed to predict the likelihood of road accidents based on historical traffic and environmental data. The goal is to identify high-risk conditions and improve road safety awareness.

🎯 Objectives

Predict whether an accident is likely to occur.

Handle imbalanced datasets using SMOTE.

Compare multiple ML models to select the best-performing one.

Evaluate model performance using classification metrics.
